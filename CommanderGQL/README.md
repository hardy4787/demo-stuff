## Run the project:

To run the project you need to first install **Docker** and **.NET 6.0 SDK** 

docker for windows : https://docs.docker.com/desktop/install/windows-install/
.NET :  https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/sdk-6.0.403-windows-x64-installer

then you need to run your Docker and to cd CommanderGQL/
Then to run sql server container by typing: 
docker-compose up -d

run .net application as well:
dotnet run

and use postman/insomnia and so on to test api
